# Files_old - Archivos y scripts obsoletos

Esta carpeta contiene archivos que ya no se usan en el sistema actual pero se conservan por referencia histórica.

## Contenido

- **config/**: Configuración de Replit (.replit)
- **docs/**: Documentación desactualizada (Replit, Cloudinary, Neon)
- **scripts/**: Scripts de migración, sincronización y fixes puntuales ya ejecutados
- **objectStorage.ts**: Cliente de Replit Object Storage (reemplazado por Supabase)
- **cloudinary-mapping.json**: Mapeo de imágenes Cloudinary (reemplazado por Supabase)
- **sed0r2hho**: Archivo residual de Replit

## Sistema actual

- **Base de datos:** Supabase (PostgreSQL)
- **Storage:** Supabase Storage
- **Deploy:** Railway
